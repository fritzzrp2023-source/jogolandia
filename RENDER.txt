Deploy gratuito da Jogarium no Render

1. Coloque esta pasta em um repositorio no GitHub.
2. Entre em https://dashboard.render.com
3. Clique em New > Web Service.
4. Conecte o repositorio da Jogarium.
5. Use estas configuracoes:

Name: jogarium
Language: Node
Build Command: npm install
Start Command: npm start
Health Check Path: /api/health

6. Em Environment Variables, coloque:

HOST=0.0.0.0
PUBLIC_URL=https://sua-url.onrender.com
SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_SERVICE_ROLE_KEY=sua-chave-service-role

7. Depois que o deploy terminar, copie a URL do Render, algo como:

https://jogarium.onrender.com

8. Confira em Environment Variables:

PUBLIC_URL=https://jogarium.onrender.com
SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_SERVICE_ROLE_KEY=sua-chave-service-role

9. Clique em Manual Deploy > Deploy latest commit.

Importante sobre banco de dados:

O projeto usa Supabase como banco de dados. Por isso, mantenha estas variaveis no Render:

SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_SERVICE_ROLE_KEY=sua-chave-service-role

Sem essas duas variaveis, cadastro e login nao conseguem salvar nem buscar usuarios.
